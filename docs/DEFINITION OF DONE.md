# Definition of Done

Before anything is moved to the “DONE” column, please make sure:

1. All code has been tested, where possible, and when necessary.

2. Your working branch has had a pull request submitted, and accepted, to merge back to the feature branch.

3. The feature branch has had a pull request submitted, and accepted, to merge back into the development branch.

4. All merge conflicts have been resolved at each pull request stage.
